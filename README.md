# Login Authentication System in Flask

## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
git clone https://github.com/Xamar1n/kti-flask
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python app.py
```

## Viewing The App

Go to `http://127.0.0.1:5000`
